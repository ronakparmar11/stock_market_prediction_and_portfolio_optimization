# Stock Market Prediction & Portfolio Optimization System

AI-powered financial analytics platform that predicts stock prices 
using machine learning and optimizes portfolios using Modern Portfolio Theory.

## Features
- Stock price prediction using ML models
- Portfolio optimization using Sharpe Ratio
- AI chatbot for financial insights
- Interactive financial dashboard
- Candlestick charts and analytics

## Tech Stack
Frontend: Next.js, React, Tailwind  
Backend: FastAPI, Python  
ML: Scikit-learn, Pandas, NumPy  
AI: Llama 3.1 via Featherless API  

## Project Structure
backend/
frontend/

## Run Backend
pip install -r requirements.txt
uvicorn main:app --reload

## Run Frontend
npm install
npm run dev
